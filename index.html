Data Interactives

Concord Consortium's CODAP data interactives repository: github.com/concord-consortium/codap-data-interactives

Concord Consortium's CODAP project repository: github.com/concord-consortium/codap

To find out more about the CODAP project: About CODAP


This site has a list of interactives that can be used with CODAP.

You can click on Launch to launch the interactive in CODAP.
You can also drag Link into an open CODAP document to add the interactive into the document.


These interactives show how CODAP is used with our partners' models and data.

These interactives involve playing games Playing computer games creates a lot of data, which usually disappear when the game is over. But in Data Games, you learn to analyze data that are saved, learning math and data skills that help you win!
